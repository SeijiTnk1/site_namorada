# ❤️ Projeto para Minha Namorada

Um site interativo desenvolvido como uma forma especial de guardar memórias, momentos e sentimentos em um único lugar.

## 📖 Estrutura do Projeto

O site é composto por 4 páginas principais, cada uma com um propósito especial:

* **Home:** página inicial com a apresentação do projeto e uma mensagem de boas-vindas.
* **For You:** homenagem criada para o aniversário dela, com textos e mensagens especiais.
* **Our Day:** página dedicada ao Dia dos Namorados, celebrando momentos e sentimentos compartilhados.
* **Memories:** uma coleção de fotos, vídeos e recordações importantes da nossa história juntos.

Todas as páginas foram desenvolvidas para criar uma experiência visual personalizada, combinando animações, música e elementos interativos para tornar cada memória ainda mais especial.


## 🛠️ Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript
* Git
* GitHub Pages

## 📸 Funcionalidades

* Cabeçalho de navegação entre as páginas
* Navegação entre páginas
* Timeline de memórias
* Reprodução automática de vídeos
* Player de música personalizado
* Controle de play, pausa e troca de músicas
* Efeitos visuais e animações
* Layout adaptado para desktop e mobile
